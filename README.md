# Azure Entra ID — IAM Lab Portfolio

A collection of hands-on Identity & Access Management (IAM) labs performed in Microsoft Azure Entra ID. This repository documents real-world IAM configurations including Administrative Units, Conditional Access Policies, Role Assignments, and more.

> **Author:** Masrur Ahmed  
> **Platform:** Microsoft Azure / Entra ID  
> **Focus:** Identity Governance, Access Control, Zero Trust Principles

---

## Labs

| # | Lab Title | Key Concepts |
|---|-----------|--------------|
| 01 | [Creating a Restricted AU for NJ & Assigning User Admin Role](./lab-01-nj-administrative-unit/README.md) | Administrative Units, Scoped RBAC, Role Assignment |

## Skills Demonstrated

- Creating and managing **Administrative Units** in Microsoft Entra ID
- Implementing **scoped Role-Based Access Control (RBAC)**
- Delegating admin responsibilities without over-provisioning permissions
- Following **least privilege** access principles
